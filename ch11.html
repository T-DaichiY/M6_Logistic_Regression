🧮 第11章：バイナリ予測変数②：係数の解釈

係数は「対数オッズの変化量」= log(オッズ比)。ForeignWorkerの係数を3ステップで手計算して再現する。

Unit 1Unit 2Unit 3
01

バイナリ係数の意味

係数 = 対数オッズの変化量

バイナリ予測変数の係数は「そのバイナリ変数が0→1に変化したときの対数オッズの変化量」です。単純な確率の差ではなく、オッズ比の対数になります。

📐
係数 β₁ = log(odds|X=1) − log(odds|X=0)
= log(odds|X=1 / odds|X=0)
つまり「2つの状態のオッズ比」の対数

重回帰の係数が「Xが1増えたときのYの変化量」だったのと同様に、ロジスティック回帰の係数は「Xが1増えたときの対数オッズの変化量」です。

02

手計算で係数を再現する

3ステップで係数を導出

ForeignWorkerの係数（−1.297）を手計算で導出します。

# ForeignWorker係数の解釈：ステップごとに計算# Step1: FW=0（外国人）のときp0 <- 0.891 # P(Good|FW=0)odds0 <- p0 / (1 - p0) # → 8.25lo0 <- log(odds0) # → 2.11（切片）
# Step2: FW=1（国内労働者）のときp1 <- 667/963 # P(Good|FW=1) ≈ 0.693odds1 <- p1 / (1 - p1) # → 2.255lo1 <- log(odds1) # → 0.813# Step3: 係数 = log(新オッズ / 旧オッズ)coef_fw <- log(odds1 / odds0) # → -1.297 ← 係数と一致！
# Rで確認coef(mod_fw)["ForeignWorkeryes"] # → -1.2970# オッズ比に変換（解釈しやすい形）exp(coef(mod_fw)) # ForeignWorkeryesのオッズ比 → 0.273# → 国内労働者の良い信用オッズは外国人の0.273倍（外国人の方が有利）
ポイント：係数を求めるのは「引き算」ではなく「オッズ比の対数」。
対数オッズで差をとる = 元のスケールでオッズ同士を割り算することと同じ。
exp(係数) = オッズ比 = 0.273 →「国内労働者の良い信用オッズは外国人の27%（＝外国人の方が3.7倍有利）」
03

係数の解釈まとめ

対数オッズ・オッズ比・確率の3段階
📌 係数＝差、exp(係数)＝比：バイナリ予測変数の係数は「X=1グループとX=0グループの対数オッズの差」です。指数を取った exp(β) は両グループのオッズ比になります。オッズ比が1なら差なし、1より大きければ増加、小さければ減少。確率の差で考えようとすると非線形で破綻するので、必ず「対数オッズ → オッズ → 確率」の順でたどるのが安全です。
