Chapter 4 · Multicollinearity & VIF

予測変数どうしが
似すぎると、
係数がおかしくなる。

「多重」＝複数、「共線」＝同じ線の上。似た変数が重なると係数が暴れ、符号まで逆転し、p値も当てにならなくなります。 相関行列では見抜けない隠れた共線性を、VIF（分散拡大係数）で検出します。

What · 多重共線性とは

「同じことを言う友達」が
複数いる状態

多重共線性とは、モデル内の2つ以上の予測変数が互いに強く相関し、強い線形関係を示す状態。 これが起きると問題が連鎖します。

問題 01

係数がうまく推定できない

推定が不安定になり、異常に大きく（inflated）なる。

問題 02

符号が直感に反する

プラスのはずがマイナスになるなど、期待と逆の符号が出る。

問題 03

標準誤差が膨らむ

それに依存するt統計量やp値が信頼できなくなる。

このクラスでは、2つの予測変数の相関の絶対値が 0.5 を超えたら共線性ありとみなします（符号は無視、大きさだけ見る）。

予測したいもの共線性が起きる予測変数なぜ似ているか（講義の例）
血圧体重・身長・BMIBMIは体重と身長から計算 → 当然相関。回帰すると「体重↑で血圧↑、BMI↓で血圧↑」のような変な符号が出ることも。
家の価格寝室の数・築年数・延床面積寝室が多い家は面積も広い → 寝室数と延床面積が相関する。
🍎 かんたん説明
多重共線性は「同じことを言う友達が2人いる」状態。クイズの答えを2人に聞いても同じことしか返ってこないなら、1人で十分。 似た情報を持つ変数を2つ入れると、モデルが混乱します👬
First check · 相関行列

相関行列は便利。でも限界あり

まず相関行列で予測変数どうしの相関を見ます。応答変数Yと強く相関するのは良いこと。 でも予測変数X1とX2どうしが強く相関していたら要注意です。

講義の偽データの例：Y を X1〜X4 で予測。X1・X2 は Y と強く相関していて期待が高まりますが、 よく見ると X1 と X2 が互いに相関 0.98。これは明確な多重共線性です（|r|>0.5）。

1# 自転車データ：数値列だけ選んで相関行列（real_feel は除外）
2bikes %>% select(rentals, humidity, temperature, windspeed) %>% cor()

自転車データでは、temperature は rentals と強く相関（良い）。予測変数どうしで |相関|>0.5 のペアは無く、共線性の問題は無さそうでした ✅

⚠️ 相関行列の落とし穴
相関行列は2つの変数のペアしか調べられません。3つ以上が一緒に絡み合った共線性は見逃します。 実際、偽データでは相関行列で見えたのは X1–X2 の 0.98 だけ。でも本当は X4 も X1・X2 と絡んでいたのに気づけませんでした。だから次のVIFが必要です。
🍎 かんたん説明
相関行列は「2人組のケンカ」しか見つけられない先生。「3人グループで結託してる」のは見抜けない。 3人以上のグルを暴くには、もっと賢い探知機（VIF）が必要です🕵️
Interactive · VIFゲージ

変数が似るほど、
VIFは爆発的に膨らむ

VIF＝1 / (1 − R²)。ここでの R² は「ある予測変数を、他の予測変数で予測したときの R²」。 たとえば temperature ~ humidity + windspeed の R²。気温を他の変数で説明できてしまう（R²が高い）なら、気温は他と被っている＝VIFが大きくなります。 VIFの逆数が許容度（Tolerance）です。スライダーで“2変数の似ぐあい”を上げてみてください。

📌 VIF>5（＝Tolerance<0.2）で共線性あり・対処すべき。VIF>10（Tolerance<0.1）で非常に深刻。

許容度（Tolerance）のほうが直感的

許容度は「ある予測変数の分散のうち、他の変数で説明されない割合」。講義の例：

許容度 = 0.88

  • ✅ 良い。その変数は他にない情報（シグナル）を多く持つ
  • 残すべき

許容度 < 0.2

  • ⚠️ 他の変数とほぼ同じ情報
  • 無くても他が説明できる → 不要かも
🍎 かんたん説明
VIFは「この子の発言、どれくらい他の子とかぶってる？」メーター。VIFが大きい＝かぶりまくり。 許容度はその逆で「どれくらいオリジナルな意見を持ってるか」。オリジナルが多いほど、チームに必要な人材です🗣️
In R · 計算する

car または olsrr パッケージで

1# car パッケージの vif()
2library(car)
3vif(bikes_model_1) # 全部 5未満 → 共線性なし ✅
4ols_vif_tol(bikes_model_1) # olsrr：VIFと許容度を同時に

自転車モデルの3変数はすべて VIF<5・許容度>0.2 ＝ 共線性なし。たとえば湿度は他の変数で説明されない独自の情報を多く持つので、残すべきです。

偽データでの例（共線性あり）
X1・X2 の VIF は5超。とくに X2 の VIF は 115 と巨大で深刻。さらに相関行列では見えなかった X4 も巻き込まれていたと判明。これがVIFの威力です。
Fix · 見つけたらどうする

削るか、まとめるか

対処法1：1つを削除

絡み合った変数のうち1つだけ残して他を削る。偽データでは一番VIFが高いX2（VIF=115）を削除。 再計算すると残りはすべて5未満に。X4とX1の問題も同時に解消しました。

対処法2：まとめる

相関する変数を合体させて新しい1変数に。 例：「延床面積」と「階数」を掛けて「総床面積」に。偽データなら X1 と X2 の積や商で新変数 X5 を作る、など。

手順（ループ）

VIFを計算する
VIF > 5 の変数があるか確認する
一番VIFが高い1つを削除（または合体）する
全部 VIF < 5 になるまで繰り返す
📌 さらに高度な手法
主成分分析（PCA）やLassoなどの正則化もありますが、これらはこの講義の範囲外です。
🍎 かんたん説明
同じことを言う友達が3人いたら、1人だけ残せばOK（対処法1）。あるいは3人の意見を「みんなの総意」として1つにまとめる（対処法2）。 どちらもダブった情報を整理してスッキリさせるのが目的です🧹

Glossary · 重要用語

この章の用語

多重共線性
Multicollinearity
予測変数どうしが強く相関し係数が不安定になる状態。
VIF
Variance Inflation Factor
1/(1−R²)。R²は他の予測変数からその変数を予測した値。5超で共線性。
許容度
Tolerance
VIFの逆数。他で説明されない分散の割合。0.2未満で問題。
相関行列
Correlation Matrix
変数ペアの相関一覧。3変数以上の共線性は見逃す。
car / olsrr
car / olsrr packages
VIFや許容度を計算するRパッケージ。
