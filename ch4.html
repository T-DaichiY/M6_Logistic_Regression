Chapter 4

🐚 係数の解釈：アバロンの例

予測変数なしモデルの切片が対数オッズであることを、アバロンの実データで手計算しながら確認。係数→確率への変換を完全マスター。

① 切片のみのモデル② 比率から手計算③ 確率への逆変換④ 解釈のまとめ📝 Quiz
①

まず予測変数なしのモデル

切片だけのモデルが示すもの

ロジスティック回帰の係数解釈を学ぶために、まず予測変数が一切ない（切片のみ）モデルから始めます。実際の分析ではやりませんが、基礎を理解するために重要です。

# 予測変数なし（切片のみ）のモデルmodel0 <- glm(Type ~ 1, data = abalone2, family = binomial)summary(model0)
Coefficients: Estimate (Intercept) 0.1565 ← これが何を意味するか？

この係数 0.1565 は何を意味するのでしょうか？答えはオスである対数オッズです。

🍎 かんたん説明

予測変数がないと、モデルは「データ全体でオスの割合はどのくらいか？」だけを見ています。それが対数オッズの形で係数に現れます。

②

比率から係数を手計算で再現

1307匹と1528匹から出発

アバロンデータには（不明を除いた）メス1,307匹・オス1,528匹がいます。ここからステップごとに確認しましょう。

# データの件数確認table(abalone2$Type)# F M # 1307 1528# Step 1: 確率（オスの割合）P_male <- 1528 / (1307 + 1528)# → 0.5387（約53.9%がオス）# Step 2: オッズodds_male <- P_male / (1 - P_male)# → 1.169# Step 3: 対数オッズlog_odds_male <- log(odds_male)# → 0.1565　← model0の切片と完全一致！✅
P_male = 0.5387 odds_male = 1.169 log_odds_male = 0.1565 ← glmの係数と一致
📌 ポイント：予測変数がないとき、切片 = そのカテゴリ（ここではオス）の単純な対数オッズです。
③

係数から確率に「逆変換」する

exp() と確率の計算

係数（対数オッズ）から元の確率に戻す手順：

# 切片を取り出すcoef0 <- coef(model0)["(Intercept)"]# → 0.1565# Step 1: 対数オッズ → オッズ（exp()で指数化）odds_from_coef <- exp(coef0)# → 1.169# Step 2: オッズ → 確率p_from_odds <- odds_from_coef / (1 + odds_from_coef)# → 0.5387（元の確率に一致！）✅
切片(log odds) = 0.1565 exp(切片) = 1.169 (オッズ) 確率 = 0.5387 ← 1528/(1307+1528) と一致
🍎 かんたん説明

係数を見てもすぐには意味がわからない（対数オッズだから）。exp()でオッズに戻し、さらに odds/(1+odds) で確率に直す。この2ステップが「係数の解釈」の基本です🗝️

④

解釈のまとめと次のステップ

予測変数を追加するとどうなるか

切片のみのモデルでわかったこと：

📊
切片 = 対数オッズ
予測変数が0のときの log odds
🔢
exp(切片) = オッズ
指数化で元のオッズに戻る
📐
確率への変換
P = odds / (1 + odds)
⚠️ 次の章以降：予測変数（例：diameter）を追加すると、切片の意味が変わります。「その予測変数がゼロのときの対数オッズ」になります。係数の解釈には注意が必要です。
🔵 覚えておくべき式：
対数オッズ → オッズ：exp(係数)
オッズ → 確率：P = odds / (1 + odds)

📖 用語集

切片
Intercept
予測変数がゼロのときの対数オッズ。予測変数なしモデルでは単純な対数オッズ。
exp()
Exponential Function
eのx乗を計算する関数。対数オッズをオッズに変換する。
オッズ比
Odds Ratio
2つのオッズの比率。係数をexp()した値。
対数変換
Log Transformation
自然対数を取ること。オッズを対数オッズに変換する際に使う。
