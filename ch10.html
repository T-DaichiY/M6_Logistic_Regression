Chapter 10

🔀 第10章：バイナリ予測変数①：切片の解釈

予測変数を追加すると切片の意味が変わる。ForeignWorkerを例に「条件付き切片」を手計算で再現する。

Unit 1Unit 2
01

バイナリ予測変数を追加する

ForeignWorkerをGermanCreditモデルに追加

第9章では切片のみのモデルを確認しました。ここではバイナリ変数（0/1の2値変数）ForeignWorkerを予測変数として追加します。

⚠️ contrasts()の確認が必須！
このデータでは ForeignWorker: 0=yes（外国人）、1=no（国内労働者） という直感と逆のコーディングになっています。解釈前に必ず確認してください。
# ForeignWorkerを予測変数として追加mod_fw <- glm(Class ~ ForeignWorker, data=GermanCredit, family=binomial)summary(mod_fw)# 注: ForeignWorker: 0=yes(外国人), 1=no(国内)# contrasts()で必ず確認！
係数がマイナスであることに注目。ForeignWorkerが0→1に増加（つまり外国人→国内労働者）すると、良い信用の対数オッズが減少する。つまり外国人労働者の方が良い信用の確率が高い。
02

切片の解釈（バイナリ予測変数追加後）

切片は「追加した予測変数=0のときの対数オッズ」

予測変数を追加すると、切片の意味が変わります。以前は「全体の対数オッズ」でしたが、バイナリ予測変数を加えると「その予測変数=0のときの対数オッズ」になります。

📐
切片の新しい意味：
logit(P) = β₀ + β₁×X₁ において、X₁=0 のとき
logit(P) = β₀ のみ → 切片だけが残る
例：ForeignWorker=0（外国人）の場合の対数オッズ
# 切片の解釈（ForeignWorker=0のとき）intercept <- coef(mod_fw)[1] # → 2.1102odds_fw0 <- exp(intercept) # → 8.25prob_fw0 <- odds_fw0 / (1 + odds_fw0) # → 0.891# 確認：ForeignWorker=0の行の実際の比率prop.table(table(GermanCredit$ForeignWorker, GermanCredit$Class), 1)# Good比率: 33/37 ≈ 0.891 ← 一致！
切片=2.11 → exp(2.11)=8.25（オッズ）→ 8.25/9.25=0.891（確率）。実際のデータでForeignWorker=0の行のGood比率（33/37≈0.891）と一致します。

📖 用語集

バイナリ予測変数
0または1の2値をとる予測変数。例：ForeignWorker（0=外国人、1=国内）
条件付き切片
バイナリ予測変数=0のときの対数オッズ。単純な全体の対数オッズとは異なる
contrasts()
Rでカテゴリ変数のコーディング（参照カテゴリ）を確認する関数
オッズ比
exp(係数)。予測変数が1増えたときのオッズの倍率
