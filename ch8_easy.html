🍎 第8章 超やさしい版：モデル評価③：Z値とR²

Z値 = 係数÷標準誤差。電卓で自分で計算・Rコードで確認。R²がない理由と代替指標も学ぼう。

一言でいうとZ値を自分で計算RでZ値確認・疑似R²📏 本編はこちら
⚡

一言でいうと

個別の変数はZ値で判定。R²はないので逸脱度・AICで代用。

🎯 これだけ覚えればOK

重回帰のT値の代わりにZ値を使う。計算式は同じ：Z = 係数 ÷ 標準誤差。|Z|>1.96でp<0.05（有意）。R²は最尤推定と相性が悪いので存在しない。代わりに逸脱度・AIC・疑似R²を使う。

📈 重回帰
個別検定：T値
全体評価：R²（決定係数）
比較：調整済みR²・AIC
📐 ロジスティック回帰
個別検定：Z値
全体評価：逸脱度（Deviance）
比較：AIC
①

Z値を自分で計算してみよう

係数 ÷ 標準誤差 = Z値

入力フォームに係数（β）と標準誤差（SE）を入れると、Z値と有意性がリアルタイムで表示されます。

📏 Z値 電卓
Z = 係数 ÷ 標準誤差
-21.547
—
📌 summary()に表示される「z value」はこれと同じ計算。|Z|>1.96ならp<0.05で有意（95%信頼区間内にゼロが含まれない）。
②

RでZ値確認・疑似R²を計算

summary()の読み方と代替R²

▶ summary()でZ値を確認する

# summary()でZ値を確認
model <- glm(Sex ~ Diameter, data=Abalone, family=binomial)
summary(model)
#             Estimate Std. Error z value Pr(>|z|)
# (Intercept)   -7.218      0.335  -21.57  <2e-16 ***
# Diameter       21.99      0.924   23.80  <2e-16 ***

▶ Z値を手計算で再現する

# Z値を手計算
coef_est <- -7.218
std_err  <- 0.335
z_val    <- coef_est / std_err
z_val  # → -21.55（summary()の-21.57と一致）
# |Z| > 1.96 → p < 0.05 → 有意
abs(z_val) > 1.96  # → TRUE

▶ R²の代わりにMcFaddenの疑似R²を計算

# ロジスティック回帰にR²はない
# 代わりに McFadden の疑似R²を使うことがある
null_dev  <- model$null.deviance
resid_dev <- model$deviance
mcfadden_r2 <- 1 - (resid_dev / null_dev)
mcfadden_r2  # 0〜1（大きいほど良い）
🍎 ひとことで

R²は「OLS（最小二乗法）専用の評価指標」。ロジスティック回帰はMLE（最尤推定）を使うため定義が合わない。疑似R²という代用指標があるが、R²とは意味が異なるので注意。

🤔 答えられるかな？（クリックで答えが開く）

Q1. Z値の計算式は？A. Z = 係数（β） ÷ 標準誤差（SE）。重回帰のT値と同じ計算式。
Q2. |Z|>1.96は何を意味する？A. p値<0.05（有意水準5%）で統計的に有意。その予測変数が0でない可能性が高い。summary()の***はこれ以上に有意。
Q3. ロジスティック回帰にR²がない理由は？A. 最尤推定（MLE）を使うためR²が定義されない。代わりに逸脱度・AIC、または疑似R²（McFadden等）を使う。
