📈 第12章：連続予測変数の解釈

Ageを予測変数として追加。係数は全年齢範囲で一定のオッズ比の対数。手計算でバイナリと同じ方法で再現する。

Unit 1Unit 2Unit 3
01

連続予測変数の追加

Ageを予測変数として追加する

バイナリ変数の次は、連続変数（Continuous Predictor）の解釈を学びます。GermanCreditデータに年齢（Age）を追加します。

良い信用を持つ人の年齢をグループ別に確認すると、Good信用の人の方がわずかに年齢が高い傾向があります。

# Ageを連続予測変数として追加mod_age <- glm(Class ~ Age, data=GermanCredit, family=binomial)summary(mod_age)# Age係数 ≈ 0.0184（1歳増加→対数オッズが0.0184増）
連続変数の係数の解釈：「Ageが1単位増加するごとに、対数オッズが0.0184増加する」。これはAgeの全範囲で一定（線形な変化）。
02

係数を手計算で再現する

バイナリ変数と同じ「オッズ比の対数」

連続変数の係数も、バイナリ変数と同じ方法で求められます。「X=0のときのオッズ」と「X=1のときのオッズ」を計算し、その比の対数をとります。

# Age=0のとき（仮想値：仕組みの確認用）b0 <- coef(mod_age)[1] # 切片 ≈ -0.354b1 <- coef(mod_age)[2] # Age係数 ≈ 0.0184p_age0 <- plogis(b0 + b1*0) # → 0.4124odds_a0 <- p_age0 / (1 - p_age0) # → 0.7014# Age=1のときp_age1 <- plogis(b0 + b1*1) # → 0.4156odds_a1 <- p_age1 / (1 - p_age1) # → 0.7143# 係数の再現：log(オッズ比)log(odds_a1 / odds_a0) # → 0.0184（係数と一致！）
🔁
どの2値でも同じ！
Age=55→56でも、Age=33→34でも
log(odds_next / odds_prev) = 0.0184（係数と一致）
これが「線形性」の意味。係数は全範囲で一定。
03

オッズ比と複合モデル

exp(係数)の解釈とFull Modelへの拡張
📌 連続変数の一定性の仮定：連続予測変数の係数は「1単位増えるごとの対数オッズの一定の変化量」です。だから exp(β) はどの値の区間でも同じオッズ比になります（年齢55→56でも0→1でも同じ）。ただし確率の増え方は一定ではなく、シグモイドの形ゆえ中央付近で大きく、両端で小さくなります。「対数オッズは一定、確率は一定でない」が連続変数解釈の核心です。
# exp(係数) = オッズ比（連続変数）exp(coef(mod_age)["Age"]) # → 1.0186# 解釈：1歳増えるごとにオッズが1.0186倍（約1.86%増加）# ForeignWorkerとAgeを組み合わせたフルモデルmod_full <- glm(Class ~ ForeignWorker + Age, data=GermanCredit, family=binomial)AIC(mod_fw, mod_age, mod_full) # 最良モデルを選ぶ

📖 用語集

連続予測変数
Continuous Predictor
年齢・収入など連続した数値をとる予測変数。係数は1単位増加あたりの対数オッズ変化を示す。
オッズ比（連続）
Odds Ratio (Continuous)
exp(係数)。予測変数1単位増加あたりのオッズの倍率。
線形性
Linearity Assumption
連続変数の係数は全範囲で一定（Age=0→1でもAge=55→56でも同じ変化量）。
フルモデル
Full Model
複数の予測変数を全て含むモデル。AICで他のモデルと比較して最適なものを選択する。
