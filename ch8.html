Chapter 8

📏 モデル評価③：Z値とR²

個別の予測変数はZ統計量（係数÷SE）で評価。ロジスティック回帰にR²は存在しない理由と代替指標を理解する。

① Z統計量とは② Rで手動計算③ R²は存在しない📝 Quiz
①

個別の予測変数：Z統計量

T検定の代わりにZ検定

重回帰では個別の予測変数の有意性をT統計量で確認しました。ロジスティック回帰では代わりにZ統計量（Wald統計量）を使います。

Z = 係数（β̂）÷ 標準誤差（SE）

これは統計学の基本から来ており、見た目は全く同じです。「係数がゼロから何標準誤差分離れているか」を測ります。

🍎 かんたん説明

Z値は「係数が偶然ゼロになった可能性はどのくらいか」を示します。Z値が大きい（絶対値）ほど、その予測変数は有意です。重回帰のT値と全く同じ発想です🎯

②

RでZ値を手動計算

係数と標準誤差を取り出す

Z値はsummary()の表に表示されますが、手動で計算して確認してみましょう。

# モデルの係数テーブルを取り出すcoef_table <- summary(model1)$coefficients# Diameterの係数（2行目）b_diameter <- coef_table[2, 1] # Estimatese_diameter <- coef_table[2, 2] # Std. Error# Z値を手計算z_val <- b_diameter / se_diameterz_val # → −5.168（summaryの値と一致）✅
b_diameter = −2.847 se_diameter = 0.551 z_val = −5.168 ← summary()と完全一致
📌 負のZ値：Diameterが大きい → オスである対数オッズが下がる（負の係数）。符号は関係の方向を示します。
③

ロジスティック回帰にR²はない

代替指標を使う
📌 擬似R²の中身：線形回帰のR²は「説明された分散の割合」ですが、二値の結果には分散の分解が当てはまりません。代わりにMcFaddenの擬似R²（1 − LL_model/LL_null）やNagelkerkeの指標を使います。これらは対数尤度の比から作られ、0〜1の目安にはなりますが、線形回帰のR²ほど直接「当てはまりの良さ%」とは読めない点に注意します。

重回帰で最重要だったR²（決定係数）は、ロジスティック回帰には存在しません。なぜなら、ロジスティック回帰は最小二乗法ではなく最尤推定（MLE）を使うためです。

指標重回帰ロジスティック回帰
全体的な当てはまりR²、調整済みR²逸脱度（Deviance）
モデル比較AIC、BICAIC、BIC（同じ）
個別予測変数T統計量Z統計量
疑似R²—McFadden's R²（参考程度）
⚠️ 疑似R²（Pseudo R²）について：Nagelkerke R²やMcFadden R²など「R²もどき」の指標が存在しますが、本物のR²ではありません。解釈は慎重に行いましょう。
🍎 かんたん説明

R²がないことを悲しまないで。逸脱度とAICがちゃんとその役割を果たしてくれます。ロジスティック回帰の世界では「これが当たり前」です🤝

📖 用語集

Z統計量
Z-statistic / Wald statistic
β̂ / SE。個別の予測変数の有意性を検定する統計量。
最尤推定
Maximum Likelihood Estimation (MLE)
ロジスティック回帰の係数推定法。観測データが最も起こりやすくなるパラメータを求める。
疑似R²
Pseudo R²
McFadden R²やNagelkerke R²など。参考程度の指標で、本物のR²とは異なる。
標準誤差
Standard Error (SE)
係数推定値のばらつきを示す値。Z = 係数 ÷ SEで使用。
