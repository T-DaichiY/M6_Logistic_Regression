連続変数Ageの係数0.0184。バイナリと全く同じ計算で再現でき、どの年齢でも一定（線形性）。

一言でいうとAgeモデル実行手計算で再現フルモデルとAIC📈 本編はこちら
⚡

一言でいうと

連続変数の係数 = 1単位増加あたりのオッズ比の対数

🎯 これだけ覚えればOK

連続変数（Age）の係数0.0184は「1歳増えるごとに対数オッズが0.0184増加」。exp(0.0184)=1.0186がオッズ比（1歳増ごとにオッズが1.0186倍）。バイナリ変数と同じ計算で手計算でも再現できる。重要：この比率は全年齢範囲で一定（線形性）。

バイナリ変数（0か1）と違い、連続変数は無数の値を取ります。でも係数の求め方は同じです。「隣り合う2つの値でオッズ比を計算し対数を取る」だけ。

バイナリ変数（復習）
X: 0→1の変化
係数 = log(odds₁/odds₀)
例：FW → −1.297
連続変数（今回）
X: n→n+1の変化（どの値でも同じ）
係数 = log(oddsₙ₊₁/oddsₙ)
例：Age → 0.0184
①

Ageのモデルを実行する

係数・オッズ比の確認

▶ Ageモデルの実行とオッズ比

# 連続変数Ageのモデルmod_age <- glm(Class ~ Age,            data=GermanCredit, family=binomial)coef(mod_age)  # Age係数 ≈ 0.0184# → 1歳増えるごとに対数オッズが0.0184増加exp(coef(mod_age)[2])  # → 1.0186（オッズ比）

②

係数を手計算で再現する

バイナリと全く同じ計算

▶ Age=0→1 で係数を手計算

# Age=0→1 でオッズ比を確認（係数を手計算で再現）b0 <- coef(mod_age)[1]   # 切片b1 <- coef(mod_age)[2]   # Age係数p0 <- plogis(b0)            # Age=0のときp1 <- plogis(b0 + b1)       # Age=1のときodds_ratio <- (p1/(1-p1)) / (p0/(1-p0))log(odds_ratio)             # → 0.0184（係数と一致！）
🍎 重要ポイント

Age=55→56でも、Age=30→31でも、log(オッズ比)は必ず0.0184になります。これが連続変数の「線形性」。任意の隣り合う2値でコードを実行して確認してみてください。

③

フルモデルとAIC比較

ForeignWorker + Age を組み合わせる

▶ 3モデルをAICで比較

# モデル比較：FW単独 vs Age単独 vs フルモデルmod_fw   <- glm(Class ~ ForeignWorker, data=GermanCredit, family=binomial)mod_full <- glm(Class ~ ForeignWorker + Age,             data=GermanCredit, family=binomial)AIC(mod_fw, mod_age, mod_full)  # 最小AICのモデルを選ぶ

🤔 答えられるかな？（クリックで答えが開く）

Q1. 連続変数の係数0.0184は何を意味する？A. 1単位増加（1歳増）あたりの対数オッズの変化量。exp(0.0184)=1.0186がオッズ比（1.86%/年の増加）。
Q2. Age=55→56と Age=30→31で対数オッズの変化は違う？A. 同じ。連続変数の係数は全範囲で一定（線形性）。どの隣り合う2値でも log(オッズ比) = 0.0184。
Q3. FW + Ageのフルモデルをどう評価する？A. AIC(mod_fw, mod_age, mod_full)で比較。最もAICが低いモデルを選ぶ。逸脱度も下がっているか確認する。
