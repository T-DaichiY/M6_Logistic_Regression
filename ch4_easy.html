🍎 第4章 超やさしい版：係数の解釈：アバロンの例

切片が対数オッズである理由をアバロンの実データとRコードで手計算。ボタンを押してステップを1つずつ追おう。

一言でいうとRでモデル実行計算ステップを追う🐚 本編はこちら
⚡

一言でいうと

切片 = 「全体の対数オッズ」 → exp → オッズ → 確率

🎯 これだけ覚えればOK

予測変数がないとき（切片のみのモデル）、切片はデータ全体の対数オッズ。exp(切片)でオッズ、さらにodds/(1+odds)で確率に変換できる。アバロンの例では切片0.1565 → exp → 1.169 → 53.9%（オスの割合）と完全に一致。

アバロン（アワビ）のデータで、切片が何を意味するのかを手計算で確認します。これが分かれば、係数の解釈が全部わかります。

🐚 アバロンデータの概要

1,307
メス（Female）
1,528
オス（Male = 1）
2,835
合計
①

Rでモデルを実行

切片のみモデルで切片の意味をつかむ

まず予測変数を入れない「切片のみのモデル」を実行します。この切片がデータ全体の対数オッズになっているはずです。

▶ 切片のみモデル（glm with 1）

# 切片のみのモデル（予測変数なし）
model0 <- glm(Sex ~ 1, data=Abalone, family=binomial)
summary(model0)
# (Intercept) = 0.1565  ← 対数オッズ

▶ 手計算で切片を再現する

# 手計算で切片を再現する
n_female <- 1307
n_male   <- 1528
p_male   <- n_male / (n_female + n_male)
p_male  # → 0.5387
odds     <- p_male / (1 - p_male)
odds   # → 1.1685
log_odds <- log(odds)
log_odds # → 0.1557 ≈ 0.1565（切片と一致！）
②

アバロン：計算ステップを追う

ボタンを押してステップを1つずつ確認
🐚 アバロン：切片の計算ステップ
📊 データ：メス 1,307匹　オス 1,528匹（オス=1）
① 確率：P(オス) = 1528 / (1307+1528) = 0.5387（53.9%がオス）
② オッズ：0.5387 / 0.4613 = 1.169（オスの方がメスより1.169倍多い）
③ 対数オッズ：log(1.169) = 0.1565　← glm()の切片と一致！✅
④ 逆変換：exp(0.1565)=1.169 → 1.169/2.169 = 0.5387（元の確率に戻った！🔄）

▶ 係数をオッズ比に変換して解釈するコード

# 係数をオッズ比に変換して解釈
coef_val <- coef(model0)["(Intercept)"]
exp(coef_val)   # → 1.169（オッズ）
# 確率に戻す
odds_to_p <- function(odds) odds / (1 + odds)
odds_to_p(exp(coef_val))  # → 0.5387（元の確率）
📌 exp(coef(model))で係数をオッズ比に変換できます。オッズ比=1.169は「オスはメスの1.169倍のオッズがある」ことを意味します。

🤔 答えられるかな？（クリックで答えが開く）

Q1. 係数（対数オッズ）からオッズに変換するには？A. exp(係数)。例：exp(0.1565) ≈ 1.169。Rでは exp(coef(model)) で全係数を一度に変換できる。
Q2. オッズから確率に変換するには？A. 確率 = odds / (1 + odds)。例：1.169 / 2.169 ≈ 0.539（53.9%）。
Q3. 切片のみのモデルで切片が示すのは？A. データ全体での「1（オス）である確率」の対数オッズ。予測変数なし＝データの単純な比率から計算できる。

💡 まとめ

• 切片 = データ全体の対数オッズ

• exp(切片) = オッズ

• odds/(1+odds) = 確率

• 逆変換で元の確率に戻る
