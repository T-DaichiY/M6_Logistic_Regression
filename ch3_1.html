Chapter 3.1 · Influential Points

「外れ値」と
「影響点」は、
別ものだ。

外れ値でも回帰線をほとんど動かさない点もあれば、たった1点で線を大きくゆがめる点もあります。 “悪い影響を与える点”をクックの距離（Cook's D）で自動的に見つける方法を学びます。点を入れたり外したりして、線が傾く様子を確かめましょう。

Types · 外れ値の3タイプ

外れ値だからといって、
必ず悪さをするとは限らない

大事な前提：すべての外れ値が「影響点」ではありません。外れ値の現れ方には3タイプあります。

① Y方向の外れ値

Xは平均的だが Y が大きくズレている。残差が大きい。

② X方向（高レバレッジ）

Yは平均的だが X が他から大きく離れている。回帰線を自分のほうへ強く引っ張る。

③ 線の傾向に沿う

XもYも極端だが、回帰線の延長上にある。除いても線はほぼ変わらない＝影響 小。

🍎 かんたん説明
クラスの平均身長を考えるとき、すごく背が高い子（外れ値）がいても、その子が「みんなの傾向」から外れていなければ平均はあまり変わりません。 でも、たった1人だけ全然違う場所にいる子は、平均をグイッと動かす。これが「影響点」です🏫

🍎 かんたん説明
影響点を手で探すのは「砂浜から特定の貝殻を1つずつ拾って確かめる」ようなもの。大変すぎます。 だから自動で見つけてくれる「金属探知機」＝クックの距離を使うのです🏖️
Cook's D · クックの距離

点を1つ取り除いたら、
どれだけ変わる？

クックの距離は「ある観測を除いたとき、モデルがどれだけ変わるか」を測ります。 点を1つ取り出して全部を再計算し、しきい値と比べるだけ。値が大きいほど、その点の影響は大きいということです。

1# このクラスの基準: 4 / (N - K - 1) より大きければ影響点
2しきい値 = 4 / (N - K - 1) # N=観測数, K=予測変数の数

例：自転車データは N＝731行、予測変数 K＝3 なので、しきい値は 4/(731−3−1) ≈ 0.0055。これより大きいCook's Dの点を影響点とみなします。

別のしきい値（覚えやすい版）判定
D > 0.5影響点の可能性あり
D > 1.0かなり高い確率で影響点
D > 4/N予測変数の数を使わない簡易版
🍎 かんたん説明
クックの距離は「その人がいなくなったら、クラスの雰囲気がどれだけ変わるか」のメーター。 1人抜けても何も変わらない人もいれば、抜けると一気に変わる“ムードメーカー”もいる。後者が影響点です🎭
In R · 影響点を探す

OLSRR パッケージで一目瞭然

OLSRR にはCook's Dを可視化する関数があります。棒グラフで全データ点を表示し、しきい値の線を引いてくれます。

1library(olsrr)
2ols_plot_cooksd_bar(my_model) # 棒グラフ（赤=影響点, 青=正常）
3ols_plot_cooksd_chart(my_model) # もう少しきれいな版

例では、69行目がしきい値をはるかに超える巨大なCook's Dを示しました。調べてみると…

69行目（2011年3月10日）の中身
祝日ではない／3月の日／レンタル数＝たった623件（とても少ない）／湿度＝0（異常！通常の最小は0.18）／風速＝かなり速い／気温＝低め。 → モデルの予測は5,000件超。実際は623件 → 巨大な残差！
📌 分析者の視点
湿度0%は明らかに異常です。データ入力ミスの可能性を疑い、「2011年3月10日について別の情報はないか」と会社に確認するのが実務的な対応。むやみに削除せず、まず原因を考えましょう。
1# Cook's D を大きい順に上位10件
2head(sort(cooks.distance(my_model), decreasing = TRUE), 10)
3cooks_d <- cooks.distance(my_model)
4influential <- unique(which(cooks_d > 0.0055)) # 29個見つかった

本当に良くなった？を確かめる

1# 影響点を除いたデータ（先頭の −（マイナス）に注目）
2bikes_no_infl <- bikes[-influential, ]
3nrow(bikes_no_infl) / nrow(bikes) # → 0.96（96%が残る = 4%除外）

702行が残り、96%のデータを保持（4%を影響点として除外）。除去前後でモデルを比べると…

項目全データ（除去前）影響点を除去後
係数の推定値元の値変化した（影響があった証拠）
調整済みR²0.45より高い（改善！）
📌 結論
影響点を除いたら調整済みR²が上がりました。これはモデルがシグナルをよりよく捉え、ノイズを無視できるようになったこと。Cook's Dの判断通り、除去は有益でした。
🍎 かんたん説明
クラス写真で1人だけブレている子がいると全体がぼやけて見える。その子（影響点）を外して撮り直すと、みんながくっきり写る＝モデルがクリアに📸 ただし「外す前になぜブレたのか」を確認するのを忘れずに！

Glossary · 重要用語

この章の用語

影響点
Influential Point
回帰線の係数や統計量を大きく変えてしまう (X,Y) の点。
外れ値
Outlier
他から大きく離れた点。必ずしも影響点とは限らない。
高レバレッジ点
High Leverage Point
X方向に遠い外れ値。回帰線を引っ張る力が強い。
クックの距離
Cook's Distance
点を除いたときのモデル変化量。影響度の自動判定に使う。
OLSRR
olsrr package
Cook's Dの可視化など回帰診断に便利なRパッケージ。
