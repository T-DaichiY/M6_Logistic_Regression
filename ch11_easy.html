🍎 第11章 超やさしい版：バイナリ予測変数②：係数の解釈

係数は「確率の差」ではなく「オッズ比の対数」。3ステップで−1.297を手計算で再現しよう。

一言でいうと手計算で係数再現オッズ比で解釈🧮 本編はこちら
⚡

一言でいうと

係数 = 「オッズ比の対数」。引き算ではなく割り算。

🎯 これだけ覚えればOK

バイナリ係数は「P(Good|X=0)とP(Good|X=1)から求めたオッズ比の対数」。重回帰の係数が直接確率を足し引きできたのとは違い、ロジスティック回帰では「オッズ÷オッズ」の対数で表す。

FW=0（外国人）
P=0.891
オッズ=8.25
÷
log
FW=1（国内）
P=0.693
オッズ=2.255
係数 = log(2.255 / 8.25)
−1.297 ✅
①

手計算で係数を再現する

3ステップ：確率→オッズ→オッズ比→log

「なぜ係数−1.297になるのか」を実際のデータから手計算で確認しましょう。

▶ 3ステップで係数を導出

# FW=0（外国人）とFW=1（国内）のオッズを計算p0 <- 33/37          # FW=0のGood率 → 0.891p1 <- 667/963        # FW=1のGood率 → 0.693odds0 <- p0 / (1-p0)   # → 8.25odds1 <- p1 / (1-p1)   # → 2.255# 係数 = log(新オッズ / 旧オッズ)log(odds1 / odds0)        # → -1.297（係数と一致！）
📌 「係数 = 引き算では求まらない」がポイント。対数オッズ同士を引くと「log(a) − log(b) = log(a/b)」なので、実質はオッズ比の対数になります。
②

オッズ比で解釈する

exp(係数) = オッズ比 = 最も解釈しやすい形

▶ exp(係数)でオッズ比を求めて解釈

# exp(係数) でオッズ比を確認exp(coef(mod_fw))  # ForeignWorkeryesのオッズ比 → 0.273# 解釈：外国人（FW=yes=1）のオッズは#        国内労働者（FW=no=0）の0.273倍# → 国内労働者の方がGood信用になりにくい！
🍎 解釈のひとことで

オッズ比0.273 = 「外国人（FW=yes=1）の良い信用オッズは、国内労働者（FW=no=0）の27.3%」。つまり国内労働者の方が良い信用になりにくい（外国人の方がGoodになりやすい）。

🤔 答えられるかな？（クリックで答えが開く）

Q1. バイナリ係数の計算式は？A. 係数 = log(odds|X=1 / odds|X=0)。2つの状態のオッズ比の自然対数。直接確率の差ではない。
Q2. exp(係数)は何を表す？A. オッズ比。X=0とX=1のオッズの比率。0.273なら「X=1のオッズはX=0の0.273倍」。
Q3. なぜ確率を直接引き算できない？A. 係数は対数オッズのスケール。確率→オッズ→対数オッズ→係数の変換は非線形。確率の変化量は出発点によって異なる。

💡 まとめ

• 係数 = log(odds₁/odds₀)

• exp(係数) = オッズ比

• 0.273 = 国内労働者のオッズは外国人の27.3%

• 確率は直接足し引きできない
